# Assignment-1- : Git & GitHub Operations Demo

This repository demonstrates Git and GitHub workflows including:

- **Branch Management** — Creating and managing multiple branches
- **Merge Operations** — Local merges and Pull Request based merges
- **PAT & SSH Authentication** — Using Personal Access Tokens and SSH keys
- **Collaboration** — Working with collaborators via Pull Requests

## Project Structure

```
Assignment-1-/
├── README.md          # Project documentation
├── index.html         # Main HTML page
├── style.css          # Stylesheet
└── app.js             # JavaScript logic
```

## Branches

| Branch        | Purpose                              |
|---------------|--------------------------------------|
| `main`        | Production-ready code                |
| `feature-1`   | Navigation bar feature               |
| `feature-2`   | Footer section feature               |
| `dev`         | Development integration branch       |
| `hotfix`      | Bug fix branch                       |

## How to Run

1. Clone the repository
2. Open `index.html` in a browser
